# 2Dアクションゲーム
## 概要
* 某ゲームをベースにしたアクションゲームです。

## 説明
main.cpp - メイン</br>
framework.h - テンプレートの設定ヘッダ</br>
DrawBase.cpp/h - 描画物の親クラス</br>
Location2D.cpp/h - 2次元の座標クラス</br>
Vector2D.cpp/.h - 2次元のベクトルクラス</br>
MathUtility.h - 数学のユーティリティクラス</br>
SubWindow.cpp/h - サブウィンドウを出すクラス（※Win32API）</br>
